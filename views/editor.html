Variable reference

Recipient character name

Full name of the character receiving the evemail.
Example: Tiuh Shihari
Usage: {{characterName}}

Template

You can include a another template within the current template. The inclusion is recursive.
Example: -
Usage: {% my footer template %}

Last time logged off

Full timestamp of the recipient character last log off
Example: 2023-05-15T20:48:09.000Z
Usage: {{logoffDate}}

Last time logged in

Full timestamp of the recipient character last log in
Example: 2023-05-15T20:25:29.000Z
Usage: {{logonDate}}

Recipient character name

Timestamp of the recipient character creation (a character birthday)
Example: 2022-10-27T11:33:00.000Z
Usage: {{startDate}}

Sender character name

The character name that is currently logged in in Quickdraft
Example: Rosa Leaf
Usage: {{senderName}}
